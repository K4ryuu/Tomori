Setup:
	1. Drag and rop file to your server
	2. Restart or Change map
	3. Set up CFG's in cfg/Tomori/ folder
	4. Insert your SteamApi key from https://steamcommunity.com/dev/apikey to profile cfg
	5. Restart your server

Update:
	1. Drag and drop files
	2. Restart or change map
	3. (The plugin recognnize new cvars and adds it automatically)

Configurable files:
	1. cfg/Tomori/ every cfg inside the folder
	2. configs/Tomori/badwords.ini to filter bad words in chat
